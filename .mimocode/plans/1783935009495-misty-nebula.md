# Plan: Student Dashboard

## API Endpoints
- `GET /api/dashboard/stats` - KPI stats
- `GET /api/dashboard/student/topic-performance` - Topic accuracy
- `GET /api/dashboard/student/subject-performance` - Subject accuracy
- `GET /api/dashboard/student/strengths-weaknesses` - Strong/weak topics
- `GET /api/dashboard/student/performance-trends` - Score trends
