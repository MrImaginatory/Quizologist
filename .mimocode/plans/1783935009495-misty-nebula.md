# Plan: Excel Import Questions Page

## Overview
Create a separate page for bulk importing questions via Excel, with sublinks under Questions in the sidebar.

---

## Sidebar Changes
Update admin nav to have expandable Questions section:
```
Questions ▼
  ├── All Questions    (/dashboard/questions)
  └── Import Excel     (/dashboard/questions/import)
```

---

## Files to Create/Modify

### 1. `lib/api-routes.ts`
Add bulk import endpoint:
```
QUESTIONS.BULK: `${BASE_URL}/api/question/bulk`,
QUESTIONS.IMPORT_TEMPLATE: `${BASE_URL}/api/question/import-template`,
```

### 2. `lib/api.ts`
Add API methods:
```typescript
questionsApi.getTemplate: (token) => // GET blob
questionsApi.bulkImport: (questions, token) => // POST bulk
```

### 3. `components/shadcn-space/blocks/sidebar-06/app-sidebar.tsx`
Update Questions nav to have children:
```typescript
{
  title: "Questions",
  icon: FileQuestion,
  children: [
    { title: "All Questions", icon: FileQuestion, href: "/dashboard/questions" },
    { title: "Import Excel", icon: Upload, href: "/dashboard/questions/import" },
  ],
}
```

### 4. `app/(dashboard)/dashboard/questions/import/page.tsx`
New page with multi-step import flow:

**Step 1: Download Template**
- Button to download .xlsx template
- Instructions on how to fill the template

**Step 2: Upload Excel**
- File upload input (accept .xlsx)
- Parse Excel client-side using `xlsx` library

**Step 3: Preview & Validate**
- Table showing parsed questions
- Status column: Ready (green) / Error (red)
- Error messages for each failed row
- Summary: "X ready, Y errors"
- Remove button for individual rows

**Step 4: Import**
- Send valid questions to POST /api/question/bulk
- Show results: imported count, failed count
- Link to view imported questions
